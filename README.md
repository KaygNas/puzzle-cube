## Puzzle Cude
Aka Rubiks Cube.

### Demo

[online app](https://kaygnas.github.io/puzzle-cube/)

#### Instruction

press `q` `w` `e` `a` `s` `d` `z` `x` `c` to rotate any slice of the cube

press `←` `↑` `→` `↓` to rotate the entire cube
### Get Start

1. install   
`pnpm install`

2. start dev server   
`pnpm dev`
